### 🚀 Проекты

#### 🎵 Spotify Clone (Pet-проект)
> *Сложный веб-плеер с функционалом, аналогичным оригинальному Spotify.*

- **Стек:** `Nuxt 3`, `TypeScript`, `Node.js`, `MongoDB`, `DeezerAPI`
- **Особенности:**
  - Полноценное управление аудиопотоком (плей/пауза, очередь, плейлисты)
  - Система авторизации пользователей (JWT) и создание личных плейлистов
  - Интерактивный интерфейс с поиском треков и отображением "активных друзей"
  - Чистая архитектура и типобезопасность благодаря TypeScript
